Compteur de lectures de billets

Ce plugin pour Dotclear 2 permet de comptabiliser les lectures de billet.

Le code de comptabilisation des lectures est à intégrer dans les fichiers de votre thème.

Il suffit d'insérer {{tpl:postCountIncrement}} juste après <div id="p{{tpl:EntryID}}" class="post"> dans le fichier post.html.

Pour afficher le compteur de lecture d'un billet, il faut ajouter {{tpl:postCountGet}} dans le fichier post.html ou home.html selon les mêmes critères que le code de comptabilisation.
